# CI/CD

코드에서 배포까지 자동화. IaC와 함께 자동화의 핵심.

## 목차

### 개념

- [ ] CI/CD란 무엇인가
- [ ] CI vs CD vs CD (Continuous Delivery vs Deployment)
- [ ] 무중단 배포 전략 (Rolling, Blue/Green, Canary)
- [ ] 브랜치 전략과 CI/CD

### Jenkins

- [ ] Jenkins 설치 (standalone, Docker)
- [ ] Jenkins Job과 Pipeline
- [ ] Jenkinsfile 작성 (Declarative vs Scripted)
- [ ] Multi Branch Pipeline
- [ ] Jenkins 멀티 노드와 빌드 속도 최적화
- [ ] Jenkins 플러그인 관리
- [ ] Jenkins 운영 (백업, 모니터링, 보안)

### GitHub Actions

- [ ] Workflow, Job, Step 기본 구조
- [ ] 트리거와 이벤트
- [ ] 보안 (시크릿, 토큰, OIDC)
- [ ] 빌드 구현 (Java, Node, Python, Go)
- [ ] 컨테이너 이미지 빌드와 레지스트리 연동
- [ ] 캐시와 Artifact 관리
